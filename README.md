# 津軽為信の野望

津軽為信を主人公に、奥羽十国の統一を目指すブラウザ戦国シミュレーションゲームです。

## 遊び方

`index.html` をブラウザで開くと、そのまま遊べます。

GitHub Pagesで公開する場合は、このフォルダ内のファイルをリポジトリのルートに配置してください。

## 公開手順

1. GitHubで新しいリポジトリを作成します。
2. `index.html` と `README.md` をアップロードします。
3. Settings → Pages を開きます。
4. Source を `Deploy from a branch` にします。
5. Branch を `main`、フォルダを `/root` にして保存します。

公開URLの例:

`https://paz67416.github.io/tsugaru-nobunaga-game/`

## ブログ・note展開

このゲームは、次の流れで使う想定です。

1. GitHub Pagesでゲーム本体を公開
2. noteで制作意図・歴史背景・攻略・設計思想を有料記事化
3. WordPressでSEO向けの紹介記事を公開
